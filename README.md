# AutoTest

Android Auto Test
Appium과 pytest를 활용한 Android 기본 앱 자동화 테스트 예제입니다.

자동화 목적: Android 앱 테스트 자동화 학습 및 포트폴리오용
사용 기술: Python, Appium, pytest, UiAutomator2, Selenium(추후 도입 예정)

수행 내용
- Appium 환경 구성 및 Android 에뮬레이터 연결
- pytest 기반 테스트 구조 설정
- 기본 설정 앱 열기 테스트 작성
- Git을 통한 버전 관리 및 GitHub Push 완료

실행 방법
1. CMD에서 appium 입력하여 서버 실행
2. Android 에뮬레이터 연결
3. 터미널에서 pytest 입력하여 테스트 실행
